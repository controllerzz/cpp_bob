# 🤖📅 Курс C++ с Бобом — Как развивался C++ (стандарты)

> Язык C++ не стоит на месте: каждые **3 года** выходит новая версия (стандарт) с новыми возможностями. Здесь — короткий таймлайн: что и когда добавили, с маленькими примерами. Чтобы ты понимал, **откуда** взялись `auto`, лямбды, умные указатели и почему старый код выглядит иначе.

**Когда читать:** в любой момент; будет понятнее, когда уже видел `auto`, лямбды, умные указатели (README, «Основы глубже»).

🤖 Боб говорит: язык растёт, как человек — с каждой версией умеет больше. Имя стандарта = **год выпуска**: C++11 = 2011, C++14, C++17, C++20, C++23.

🛠 Какой стандарт собирать — задаёшь в `.pro`: `CONFIG += c++17` (или `c++20`). ⚠️ Новые возможности требуют **свежего компилятора**. Наш курс — в основном **C++17** плюс кое-что из C++20.

Значки: 🤖 объяснение · 🧠 теория · 🛠 практика · ✅ итог · 🔗 где в курсе.

# C++98 / C++03 — фундамент (1998–2003)

🧠 Самый первый стандарт. Здесь появилось всё базовое, на чём C++ стоит:
- **классы** и **наследование**, **шаблоны**;
- **STL** — `std::vector`, `std::map`, `std::string`;
- **исключения** (`try`/`catch`).

```cpp
std::vector<int> v;          // STL была уже тогда
v.push_back(1);
```

🤖 C++03 — это просто «починка багов» C++98, по сути та же версия.

---

# C++11 — огромный скачок (2011) 🚀

🧠 Версия, после которой говорят «современный C++». Добавили так много, что язык будто стал новым:
- **`auto`** — компилятор сам выводит тип;
- **range-based for** — `for (x : контейнер)`;
- **лямбды** — функции на месте;
- **`nullptr`** вместо `NULL`;
- **умные указатели** `unique_ptr` / `shared_ptr`;
- **move-семантика** (`std::move`, `&&`);
- **многопоточность в стандарте**: `std::thread`, `std::mutex`, `std::atomic`;
- `override`, `= default`, `= delete`, `enum class`, инициализация `{}`.

```cpp
auto v = std::vector<int>{1, 2, 3};        // auto
for (int x : v) std::cout << x << " ";      // range-based for
auto twice = [](int x){ return x * 2; };    // лямбда
int* p = nullptr;                           // nullptr
```

🔗 В курсе: [`auto`/ссылки](Курс_C++_Основы_глубже.md), [умные указатели/move](Курс_C++_ООП_продвинуто.md), [потоки/atomic](Курс_C++_Многозадачность.md).

# C++14 — мелкие, но приятные (2014)

🧠 Небольшое обновление, «полирующее» C++11:
- **`std::make_unique`** (в C++11 был только `make_shared`);
- **обобщённые лямбды** — `[](auto x){ ... }`;
- возврат типа через `auto` из функций;
- **разделители цифр** — `1'000'000`.

```cpp
auto ptr = std::make_unique<int>(42);            // make_unique
auto add = [](auto a, auto b){ return a + b; };  // лямбда с auto
int big = 1'000'000;                             // разделители читаемости
```

🔗 В курсе: [`make_unique` и RAII](Курс_C++_RAII_подробно.md).

# C++17 — солидное обновление (2017)

🧠 Много удобных вещей, которыми мы пользуемся в курсе:
- **structured bindings** — `auto [a, b] = pair;`;
- **`std::optional`** — «значение или ничего»;
- `std::variant`, `std::any`, **`std::string_view`**;
- **`std::filesystem`** — работа с файлами/папками;
- **`std::pmr`** — гибкие аллокаторы;
- `if constexpr`, вложенные namespace `a::b`.

```cpp
std::map<std::string, int> m{{"port80", 1}};
for (auto& [name, value] : m)              // structured bindings
    std::cout << name << "=" << value;

std::optional<int> found = 5;              // optional: есть значение или нет
if (found) std::cout << *found;
```

🔗 В курсе: [structured bindings (map)](Курс_C++_Основы_глубже.md), [`std::pmr`](Курс_C++_Стек_и_куча.md).

---

# C++20 — второй огромный скачок (2020) 🚀

🧠 Снова крупное обновление. Главные новинки:
- **концепты** (concepts) — понятные ограничения на шаблоны;
- **корутины** (`co_await`, `co_yield`) — приостанавливаемые функции;
- **модули** (modules) — современная замена `#include`;
- **ranges** — цепочки операций над контейнерами;
- **`std::format`** — наконец удобное форматирование строк;
- **`atomic::wait()` / `notify_one()`** — ждать без busy-wait;
- три-сравнение `<=>` («космический корабль»), `std::span`.

```cpp
#include <format>
std::string s = std::format("port {} is {}", 80, "open");  // "port 80 is open"

// концепт: функция работает только с целыми числами
template <std::integral T>
T square(T x) { return x * x; }
```

🔗 В курсе: [`atomic::wait/notify`](Курс_C++_Многозадачность.md).

---

# C++23 — свежее (2023)

🧠 Самый новый из «устоявшихся». Делает повседневный код приятнее:
- **`std::print` / `std::println`** — простой вывод без `cout`;
- **`std::expected`** — «значение ИЛИ ошибка» (альтернатива исключениям);
- `std::mdspan`, улучшения ranges, «deducing this».

```cpp
#include <print>
std::println("Hello, {}!", "world");   // Hello, world!  (без std::cout)
```

🔗 В курсе: альтернатива исключениям — упоминается в [`Исключениях`](Курс_C++_Исключения.md).

## ✅ Сводка одним взглядом

| Стандарт | Год | Главное, что добавили |
|---|---|---|
| **C++98/03** | 1998 | классы, шаблоны, STL, исключения — **фундамент** |
| **C++11** | 2011 | `auto`, лямбды, range-for, умные указатели, move, потоки — **скачок** 🚀 |
| **C++14** | 2014 | `make_unique`, обобщённые лямбды, `1'000'000` |
| **C++17** | 2017 | structured bindings, `optional`, `string_view`, `filesystem`, `pmr` |
| **C++20** | 2020 | концепты, корутины, модули, ranges, `format`, `atomic::wait` — **скачок** 🚀 |
| **C++23** | 2023 | `std::print`, `std::expected`, `mdspan` |

🤖 Два **самых крупных** обновления — **C++11** и **C++20**. Между ними (14, 17) и после (23) — улучшения поменьше, но очень полезные. Большая часть нашего курса написана на **C++17**, а кое-что (например `atomic::wait`) — уже из **C++20**.

## 📖 Словарик

| Слово | Простыми словами |
|---|---|
| **Стандарт C++** | Официальная версия языка; имя = год (C++17 = 2017) |
| **Комитет (ISO)** | Те, кто утверждает новые версии каждые ~3 года |
| **`CONFIG += c++20`** | Строка в `.pro`, чтобы собирать по нужному стандарту |
| **Концепты (C++20)** | Понятные ограничения «этот шаблон только для таких типов» |
| **Корутины (C++20)** | Функции, которые умеют приостанавливаться и продолжаться |
| **Модули (C++20)** | Современная замена `#include` |

> 🤖 Главное: C++ обновляется **раз в 3 года**, и большинство удобных вещей курса (`auto`, лямбды, умные указатели, structured bindings, `atomic::wait`) пришли из конкретных версий. Знаешь, что в какой версии — понимаешь, **что доступно** и **почему старый код другой**.
